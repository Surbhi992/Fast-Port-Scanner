# 🔐 Fast Port Scanner

A fast multi-threaded port scanner built with Python.

## 🚀 Features
- Fast scanning using ThreadPoolExecutor
- Custom port range scanning
- Simple and easy to use

## 📂 Project File
- fast_port_scanner.py

## ▶️ How to Run

```bash
python fast_port_scanner.py
```
## 📌 Example

Enter target IP or domain: 127.0.0.1  
Enter start port: 20  
Enter end port: 100  

Scan Complete!  
Open Ports: [22, 80]
